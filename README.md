# Day-Planner
 This is a simple daily planner application that allows employees with busy schedules to add and manage important events, helping them effectively manage their time. The application provides a user-friendly interface with intuitive features to create and save events for specific time blocks.
<br>
Functionality: <br>
The current day should be displayed at the top of the calendar when the planner is opened.
Scrolling down should present time blocks for standard business hours.
Each time block for the current day should be color-coded to indicate whether it is in the past, present, or future.
Clicking into a time block should allow the user to enter an event.
Clicking the save button for a time block should save the text for that event in local storage.
Refreshing the page should ensure that the saved events persist.<br>
Methods:<br>This application is built using the following technologies:<br>

HTML<br>
CSS<br>
JavaScript<br>
Local storage<br>
